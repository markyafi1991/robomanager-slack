## [1.0.0] - 2026-06-24

First stable release.

### Added
- Native Slack Assistant framework (Agents & AI Apps): dedicated AI pane, suggested prompts, and a real "is thinking..." status.
- Two-way chat (DM or @mention) plus proactive morning and end-of-day nudges.
- Guardrails as code: read-only by default, append-only writes to allow-listed folders, a code-enforced privacy wall, and no auto-commit.
- Runs locally over Slack Socket Mode (no public URL); auth via the Claude Code login or the Anthropic API.
- Optional ClickUp integration.
